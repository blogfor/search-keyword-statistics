Track, save and analyze  all incoming search terms in your WordPress website

== Description ==
Search keyword Statistics is a simple and useful plugin, which allows you to view the keywords that are being searched in your WordPress website with users details.
The plugin is easy to use. To open the plugin dashboard, just click on Keyword Statistics menu item in the admin menu. Search Keyword Statistics gives you the report of how many search result has been generated from the given key word in most updated form.

== Installation ==
1. Unzip the files and folders to the /wp-content/plugins/ directory
2. Activate the plugin through the Plugins menu in WordPress
